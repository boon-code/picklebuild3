initNodes(self, nodes, colors)

source code 

Initializes nodes list.

Will be called by the controller if a new module has been chosen to configure.

Parameters:
  • nodes - Nodes of the current modules. (Just names)
  • colors - Colors of nodes. The size of this iterable container has to be the same as the size of nodes

setTextNode(self, choice)

source code 

Sets up gui for a text-based node.

If some text-like (that means not list-like or constant) node will be selected, the controller should call this method to configure the gui to let the user enter some text or expression.

Parameters:
  • choice - The configured value (or None).

setConstNode(self, choice)

source code 

Sets up the gui to just show the current choice.

Will only show the current value and not let the user configure it's value.

Parameters:
  • choice - Current value or None if not set.
