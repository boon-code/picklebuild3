Class BasicNode

source code

object --+
         |
        BasicNode

This is the root of all nodes and covers the very basics.

All node typs are derived from this one. It covers basic features like help text, status, flags and override support.

resolveFlags(self, mod)

source code 

This method resolves flags.

Resolving means that it will be tried to find the name (self._unresolved_flags only contains names) and if the node could be found, it will be included in _flags.

Parameters:
  • mod - module node that is needed to resolve.

isOverriden(self)

source code 

This method returns if this node is overriden.

To be 'overriden' means that some other node (Only makes sense for external nodes) will contain the value of this node. Therefore this node can't be configured.

Returns:
True if this one is overriden.

addInfoSeeker(self, seeker)

source code 

This method add a node that needs the input of this node.

The seeker will be added to the _iseeker list and if this node changes somehow, all 'seeker' in the list will be notified (Their update method will be called).

Parameters:
  • seeker - The node that want to be notified if this one will be set up.

notifyInfoSeeker(self)

source code 

This method will notify all seeker objects in the list.

This method should always be called if (f.e. the value of this object changes).

Note: This method won't be called by this class. It has to be implemented by subclasses.

update(self)

source code 

This method could be called by notifyInfoSeeker.

If this node has been added to some other node as info-seeker then this node will be informed (which means that this method will be called).
