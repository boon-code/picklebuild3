Class PyParser

source code

This class is used to execute inline python code.

It parses a string and writes the resulting file to a stream.

parseString(self, data)

source code 

Parses a string and executes all inline code.

All none inline code from data will just be copied to self._dst (see constructor). All python inline tags will be replaced by their output.

Parameters:
  • data (string) - Data that will be parsed and executed.
