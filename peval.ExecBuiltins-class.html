This class emulates the builtin module.

It contains a blacklist of all objects that shall not be accessed.

Access method used by objects to retrieve a member.

This method has been overridden to hide all members except members from builtins that don't show on the blacklist.

Parameters:
  • attr_name - The attribute name that has been tried to beeing accessed.
Raises:
  • NotAllowedError - If it's tried to access a member of builtins that is on the blacklist.
