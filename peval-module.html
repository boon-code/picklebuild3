Module peval

source code

This module is about inline python code evaluation.

This module is based on an old one I used in my old buildsystem. I introduced a new class, PyParser. This class replaces the parseData method. It's a bit more flexible, which I needed to calculate the current line number if an error occurred (This should be very helpful to find errors in source code using inline python code tags).
